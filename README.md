Welcome to 30daylist.com
===============================================================================

Getting Started
---------------

    clone git@github.com:evanrkeller/Thirty-Day-List.git
    hcd Thirty-Day-List
    cp config/database.example.yml config/database.yml
    gem install bundler
    bundle

    # DB setup
    rake db:create:all
    rake db:setup

    # Run the test suites
    rake
